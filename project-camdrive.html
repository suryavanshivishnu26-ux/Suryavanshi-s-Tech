CamDrive – 4WD Smart Robot

CamDrive is a dual-power-domain ESP32-based smart robotic platform designed for stability, noise isolation and extended runtime. Built for real-world performance testing and modular expansion.

Project Overview

CamDrive integrates four BO motors driven by an L298N motor driver, an ESP32 microcontroller for control logic, ultrasonic obstacle detection, and OLED display feedback. The system uses separated motor and logic power domains to reduce electrical noise and improve system reliability.

Core Specifications

Power System

  • 2× 3.7V ~10000mAh Li-Po cells
  • Dual boost converters (XL6009 + MT3608)
  • Separated motor & logic supply
  • Common ground stabilization

Drive System

  • 4× 200 RPM BO motors
  • L298N motor driver
  • 4WD differential steering

Control & Sensors

  • ESP32 Dev Module
  • ESP32-CAM support ready
  • HC-SR04 ultrasonic sensor
  • I2C OLED display

Performance

  • 6–8 hours runtime (mixed usage)
  • ~2.5–3 kg load capacity
  • Motor noise suppression capacitors
  • Modular expansion ready

Engineering Highlights

  • Motor noise suppression using electrolytic and ceramic capacitors
  • Independent battery domains to prevent brownout resets
  • Boost converter filtering for voltage stability
  • Expandable architecture for camera integration
  • Designed for long-duration field testing

Future Upgrades

  • FPV camera module integration
  • Wi-Fi remote control dashboard
  • Autonomous navigation logic
  • AI-based obstacle classification
