NAINA V.AI

Role: Embedded Systems Developer | Year: 2025 | Status: Functional Prototype

Overview

NAINA V.AI is an ESP32-CAM based AI vision assistant. It captures images, sends them to a backend server, analyzes them using Groq’s LLaMA 4 Scout model, and displays the AI response on an OLED display.

The system focuses on reliability, Wi-Fi reconnect logic, timeout handling, and transparent serial-style debugging. It demonstrates real-world integration between edge hardware and large language models.

Technologies Used

ESP32 ESP32-CAM Groq API LLaMA 4 Scout Wi-Fi Networking OLED Display

Key Features

• Real-time image capture and AI analysis

• Automatic Wi-Fi reconnect & retry logic

• OLED response display system

• Step-by-step serial monitoring
