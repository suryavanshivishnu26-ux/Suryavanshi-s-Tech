NAINA V.AI

Role Embedded Systems Developer
Year 2025
Status Prototype - Functional
ESP32-CAM + OLED AI System

Overview

NAINA V.AI is a low-cost embedded AI vision assistant built using ESP32-CAM and ESP32 DevKit. The system captures real-world images, sends them to a backend server, and analyzes them using the Groq LLaMA 4 Scout 17B model. The AI-generated response is then displayed on an OLED screen.

The project focuses on reliability, real-time feedback, Wi-Fi reconnection logic, and transparent debugging using serial-monitor style logging. Designed as a compact AI interface, it demonstrates how edge hardware can interact with large-scale language models.

Technologies Used

ESP32 ESP32-CAM Groq API LLaMA 4 Scout Wi-Fi Networking OLED Display IoT Backend

Key Features

Image Capture & AI Analysis

Captures real-world images using ESP32-CAM and processes them via LLaMA-based AI model.

Reliable Wi-Fi Handling

Automatic reconnect logic and backend health checks to prevent system crashes.

OLED Display Output

Displays AI-generated responses directly on an I2C OLED module.

Serial Debug Monitor

Step-by-step logging for complete transparency of image transfer and AI processing.

Engineering Challenges

One major challenge was HTTP timeouts during image transfer from ESP32-CAM. This was solved by optimizing JPEG quality, reducing frame size, and implementing retry mechanisms with timeout handling.

Power stability and network reliability were also key considerations. The system includes fail-safe reconnection logic and backend validation to ensure consistent AI responses.
