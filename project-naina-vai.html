NAINA V.AI

Embedded AI Vision Assistant | 2025 | Functional Prototype
ESP32-CAM • LLaMA 4 • OLED Interface

Overview

NAINA V.AI is an ESP32-CAM based AI vision system that captures real-world images, sends them to a backend powered by Groq’s LLaMA 4 Scout model, and displays intelligent responses on an OLED screen.

The system focuses heavily on reliability — Wi-Fi retry logic, timeout handling, backend health checks, and transparent serial-style debugging for every data transfer step.

Technologies

ESP32 ESP32-CAM Groq API LLaMA 4 Scout Wi-Fi OLED I2C

Core Features

• Real-time image capture and AI response

• Automatic Wi-Fi reconnection system

• OLED output display integration

• Structured debug logging architecture
